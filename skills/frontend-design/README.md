# Frontend Design Skill

Claude Code에서 사용하는 프론트엔드 디자인 스킬입니다.
기본 AI 디자인 패턴(흰색 배경, 보라색 그라데이션, Inter 폰트)을 벗어나 독창적인 UI를 생성합니다.

## 무엇을 하는 스킬인가

Claude가 프론트엔드 작업 시 자동으로 사용합니다. 아래 특징을 가진 UI를 만들어냅니다.

- 맥락에 맞는 강렬한 디자인 방향성
- 개성 있는 타이포그래피와 색상 조합
- 임팩트 있는 애니메이션과 시각적 디테일
- 범용 AI 디자인 패턴 회피

## 사용 예시

```
"뮤직 스트리밍 앱 대시보드 만들어줘"
"AI 스타트업 랜딩 페이지 만들어줘"
"게시판 UI 만들어줘"
"두더지 게임 만들어줘"
```

## 설치 방법

```
/plugin marketplace add hdudwo/hdudwo_skills
/plugin install frontend-design@hdudwo-skills-marketplace
```
