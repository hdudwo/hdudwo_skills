# hdudwo Skills Marketplace

Claude Code 스킬 마켓플레이스입니다.

## 사용 방법

**1. Claude Code를 실행합니다 (CLI)**

**2. 마켓플레이스를 등록합니다**
```
/plugin marketplace add hdudwo/hdudwo_skills
```

**3. 스킬을 설치합니다**
```
/plugin install frontend-design@hdudwo-skills-marketplace
```

**4. Claude에게 요청합니다**
```
"Frontend Design 스킬을 사용해서 랜딩 페이지 만들어줘"
```

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| frontend-design | 독창적인 프론트엔드 UI 생성 스킬 |
